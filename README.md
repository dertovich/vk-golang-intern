# vk-golang-intern
Простой чат бот с кнопками
![image](https://github.com/dertovich/vk-golang-intern/assets/86295099/1e31eae6-c34a-4890-9717-e8413b6b6580)

## Запуск проекта

```sh
git clone https://github.com/dertovich/vk-golang-intern.git
cd vk-golang-intern
go mod download
go run .
```
